get_baseline_performance() → float[source]¶

By default calculate the F1 score of the baseline model based on test data.

Calculate the ROC/AUC value for a test set that does not contains samples from all possible classes is not supported by roc_auc_score. This is why default score differs from the binary-classification score.

fit_baseline_model(train_data: DataFrame | None = None, train_labels: Series | None = None) → BaseEstimator[source]¶

Fit a baseline model. If no data is given (default), it uses the task’s train data and creates the attribute _baseline_model. If data is given, it trains this data.

Parameters:

  • train_data (Optional[pd.DataFrame], optional) – Data to train. Defaults to None.

  • train_labels (Optional[pd.Series], optional) – Labels to train. Defaults to None.

Raises:

ValueError – If train_data is given but train_labels not or vice versa

Returns:

Trained model

Return type:

BaseEstimator
